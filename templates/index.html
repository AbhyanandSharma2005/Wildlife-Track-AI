🧠 System Architecture
🏗️

Animal Face Recognition
OpenCV head-region detection → CNN embedding extraction → Cosine similarity matching against individual animal database
⚖️
Classical ML on CNN Embeddings
256-d CNN features feed into Random Forest, SVM, KNN, Gradient Boosting & Logistic Regression for accuracy benchmarking
